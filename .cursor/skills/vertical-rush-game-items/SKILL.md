# Vertical rush — предметы и враги на треке

## Эталон: `drawRivalEgg`

Яйцо задаёт **планку качества** в этом режиме:

- **Радиальный градиент** (несколько стопов, тёплый центр → глубокий пурпур по краю).
- **Обводка** + лёгкое **внешнее свечение** (halo).
- **Мелкие частицы** внутри силуэта (дрейф по `nowMs`).
- **Дополнительный акцент** (тонкое кольцо другого цвета).

Новые объекты на треке (враги, крупные пикапы) лучше проектировать **в той же визуальной логике**: объём через градиент, а не плоский залив.

## Соперник-сперматозоид: `drawRivalSwimmer`

- **Без глаз** — только **один крупный блик** на голове (не два круга).
- **Голова** — radial gradient (светлый центр → насыщенный rose), тёмный обвод.
- **Жгутик** — широкая **ленточная** форма (bezier), отдельный **белый highlight-stroke** по изгибу; читается в движении.
- **Halo + пунктирное кольцо** — узнаваемость «враждебного яйца/клетки», созвучно лобби-овуму.

## Пикапы (good / bad / obs)

Правила **без рамок и текста** на canvas:

1. **Класс** — сила **радиальной ауры** (`drawItemAura`): бафф холоднее и чуть ярче, джанк теплее, стена приглушённее.
2. **Один герой-силуэт** на предмет: игрок должен отличить предмет **за долю секунды**.
3. **Контраст и толщина**: обводка **2–2.8px** на базовом масштабе; внутренние детали **1.2–1.6px**.
4. **Масштаб в `drawPickupOrObstacle`**: ~**1.38×** к центру (подправлять вместе с аурой).
5. **Проверка**: смотреть на **~64×64 CSS px** в движении на туннеле; если сливается с фоном — усилить силуэт или анимацию (лёгкий rotate/wobble уже есть).

Идеи усиления читаемости (примеры из кода):

- **Цинк**: длинные лучи + **второе кольцо** коротких лучей.
- **Омега**: двухтоновая капсула + **тёмный шов** по центру.
- **Цитрус**: **корка** (толстый внешний сектор) + доли.
- **Чипсы**: **красная «этикетка»** на пакете.
- **Конфета**: **дуга-блик** на леденце (не второй глаз).
- **Сода**: выше стакан, **полосы** на корпусе, крупнее пузыри.
- **Стресс-узел**: заливка центра **до** линий петель.
- **Лаптоп**: **ряды клавиш** псевдо-прямоугольниками на нижней панели.

Если процедурки мало — **спрайт** `public/rush/items/{id}.png` + `drawImage`, аура по классу сохраняется.
